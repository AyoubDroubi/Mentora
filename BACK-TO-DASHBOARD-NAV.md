## ?? Implementation Details

### Icons Used
```jsx
import { ArrowLeft, Home } from 'lucide-react';

// ArrowLeft - for top back button
<ArrowLeft className="w-5 h-5" />

// Home - for header dashboard button
<Home className="w-4 h-4" />
```

### Navigation
```jsx
import { useNavigate } from 'react-router-dom';

const navigate = useNavigate();

// Both buttons use same navigation
onClick={() => navigate('/dashboard')}
```

### Animations
```jsx
// Top button arrow animation
className="group-hover:-translate-x-1 transition-transform"

// Explanation:
// - group: parent has this class
// - hover: when parent is hovered
// - -translate-x-1: move left by 4px (0.25rem)
// - transition-transform: smooth animation
```
